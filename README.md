# Vex-Robotics-Roomba
This project is based off of an already existing school project that has already been completed. It is a roomba made from Vex parts, and is designed to push certain colored blocks out of a randomly generated space while not touching another set of blocks that are a different color. We had a time constraint, about 3 weeks, and the given parts that were available to us, which was basically every vex part your could imagine. No custom parts imitating vex parts, only other materials that could aid us in making a more functional robot were allowed.

Here are the specfic criterias:
Design, build, and test a robot that clears a space of debris.
Must incorporate at least 2 different sensors.
Avoid a designated obstacle (yellow blocks)
Stay within designated boundaries (red or blue tape)

Now, given that criteria, we, me and Soren, built this.
