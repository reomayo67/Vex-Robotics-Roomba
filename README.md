# Vex-Robotics-Roomba
This is a shell of a already existing piece of robotics, the Roomba, just rebuilt with parts from VEX. Clearly, VEX limits the amount of specialty parts need to make a roomba, but doesnt make it impossible. It was coded using python, and doesnt directly imitate a roomba. I gave it my best shot, given a certain time period and the part constraints.
